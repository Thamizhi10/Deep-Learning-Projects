# Deep Learning Projects

A collection of deep learning projects covering neural network architecture design, regularization, and transfer learning, built with TensorFlow and Keras.

## Projects

| Project | Task | Key Techniques |
|---|---|---|
| [Renewind](./01-renewind-neural-network-predictive-maintenance) | Wind turbine failure prediction | Neural network architecture tuning, Dropout, Batch Normalization, cost-sensitive evaluation |
| [HelmNet](./02-helmnet-cnn-transfer-learning) | Safety helmet detection | CNNs, transfer learning (VGG-16), data augmentation |

## Tech Stack

`Python` · `TensorFlow` · `Keras` · `scikit-learn` · `OpenCV` · `pandas` · `NumPy`
