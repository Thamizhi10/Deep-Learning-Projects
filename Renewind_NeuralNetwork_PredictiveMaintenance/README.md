# Renewind — Wind Turbine Failure Prediction

## Problem Statement
Predict wind turbine generator failure to enable predictive maintenance, minimizing costly unplanned repairs and replacements.

## Approach
Built and evaluated 25+ neural network architectures, iterating on optimizers (SGD, SGD with momentum, Adam), learning rates, hidden layers, and activation functions (ReLU, tanh). Applied Dropout and Batch Normalization for regularization. Optimized for recall over accuracy, given the asymmetric business cost of missed failures (replacement > repair > inspection).

## Result
Final model achieved 85% recall and 98% precision on the test set.

Neural network design, architecture tuning, regularization, cost-sensitive model evaluation
